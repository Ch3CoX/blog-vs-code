# Shortcuts VS code

## SideBar

* `ctrl + shit + E` : «Explorer»
* `ctrl + shit + F` : «Search»
* `ctrl + shit + G` : «Source Control»
* `ctrl + shit + D` : «Debug»
* `ctrl + shit + X` : «Extensions»
  
## Explorer

* `ctrl + N` : Crear Archivo
* `ctrl + A` : Crear Carpeta
* `ctrl + S` : Guardar Archivo
* `ctrl + W` : Cerrar Archivo Abierto
* `ctrl + F` : Buscar Archivos
